## 📌 Project Overview

This project presents an **interactive Employee PaySlip Dashboard** built using **Microsoft Excel**.
It helps analyze salary structure, deductions, and net pay, providing a clear and organized view of payroll data.

---

## 🎯 Objectives

✔ Automate payslip generation
✔ Analyze salary components (Basic, HRA, Allowances)
✔ Track deductions (Tax, PF, etc.)
✔ Calculate net salary efficiently
✔ Improve payroll transparency

---

## 🛠️ Tools & Features Used

* 📊 Microsoft Excel
* 📌 Pivot Tables
* 📈 Charts & Graphs
* 🧮 Formulas (IF, VLOOKUP, SUM, etc.)
* 🎛️ Dashboard Design & Formatting
